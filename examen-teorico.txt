1.- Qué son clases y objetos? Explique la diferencia
	La diferencia entre clases y objetos es que una clase define las características de
	un objeto, mientras que un objeto es la instancia de una clase.

2.- Cuándo recomendarías usar POST? Cuándo recomendarías usar GET?
	Post se utiliza para data sensible/privada como formularios.
	Get para pedir data como listados y cosas públicas para el usuario.

3.- Por qué usar AJAX en una web? Qué beneficios puedo darle al usuario?
	Un beneficio de Ajax es que permite que una página se modifique sin la necesidad de que se
	vuelva a cargar.
